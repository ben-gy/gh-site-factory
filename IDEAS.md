# App Ideas

Add app ideas here. The scheduled task will pick the first idea from this list, build it, and remove it. One idea per line. Be as detailed or brief as you like - the task will flesh it out.


- Interactive real-time map of NASA's Deep Space Network (DSN) — a beautiful, accessible reimagining of the existing NASA DSN Now interface (https://eyes.nasa.gov/dsn/dsn.html). Three complexes: Canberra (dishes 43, 34, 35, 36), Madrid (dishes 63, 65, 53, 54, 55, 56), Goldstone (dishes 14, 24, 25, 26). Each dish shows: spacecraft currently tracked (VGR1/Voyager 1, JWST, LUCY, CHDR, M20, SOHO etc), activity type (telemetry/tracking/command), azimuth, elevation, wind speed. Existing NASA site is functional but ugly and hard to parse — the opportunity is a visually stunning, consumer-friendly live dashboard that makes deep space communication feel awe-inspiring rather than technical. Data already publicly available via JPL/Caltech API.
- An interactive map of Australian Government structure — departments, ministers, agencies, and public bodies, with budget and staffing data. Australian version of https://machineryofgovernment.uk
