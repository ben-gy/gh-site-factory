**Date:** 2026-07-12

## Idea Source
Researched. IDEAS.md was empty. Reviewed registry.json and all ~130 ben-gy repos: Australian government-data dashboards already cover crime, hospitals, crashes, dams, donations, planning, flights, aged care, charities, childcare, NDIS, pollution, road deaths, energy (NEM), pokies, and the gender pay gap. Two clear gaps stood out — **education/schools** and **regional labour markets (unemployment)**. Confirmed via WebSearch that ACARA school data requires an application form (not autonomously buildable), whereas **Small Area Labour Markets (SALM)** publishes clean, downloadable LGA-level unemployment CSVs quarterly. Chose SALM: it fills the "economic indicators / unemployment by region" category, is highly searchable, geographic, per-capita by nature (a rate), and rich with 15 years of history.

## Data Sources
- Small Area Labour Markets (Smoothed LGA datafiles, ASGS 2025), Jobs and Skills Australia / DEWR — https://www.dewr.gov.au/employment-research/small-area-labour-markets (quarterly; smoothed unemployment level, unemployment rate %, labour force by LGA, Dec-2010 → Mar-2026, 62 quarters, 544 LGAs)
- ABS ASGS 2024 Local Government Area boundaries — https://geo.abs.gov.au/arcgis/rest/services/ASGS2024/LGA/MapServer (server-simplified GeoJSON, 566 features; 542/544 LGAs matched to a boundary)

## Architecture Decisions
- **Vanilla TS over React:** single-page, tab-switched dashboard with a Leaflet map — no need for a component framework. Smaller bundle (~60KB gzipped JS).
- **Pipeline data strategy:** SALM updates quarterly and the download URL is quarter-specific, so `collect.mjs` scrapes the resource page for the current CSV link (robust across releases). `aggregate.mjs` parses the CSV (quoted fields with embedded commas), derives state from the LGA code prefix, computes labour-force-weighted national/state aggregates, and writes lgas.json / meta.json / lga.geojson.
- **Map:** real ABS GeoJSON choropleth via Leaflet (per the "never hand-draw maps" rule). Boundaries fetched server-simplified (maxAllowableOffset) → 1.48MB, joined to SALM by LGA code.
- **No chart library:** bar charts, line charts, sparklines, histogram and heatmap are hand-rolled SVG.
- **Insights/analysis** are pure, unit-tested functions computed client-side from lgas.json.

## Notable Fix
Node's undici `fetch` consistently timed out on the DEWR server while `curl` succeeded — the server tarpits non-browser User-Agents. Fixed `collect.mjs` to send browser-like headers (Chrome UA + Accept), after which collect runs cleanly end-to-end. Also fixed a CSS bug where `.map-status { display:flex }` overrode the `[hidden]` attribute, leaving the "Loading map…" overlay stuck; added `.map-status[hidden]{display:none}`.
