## Architecture Decisions
- **Vanilla TS over React**: Single-page dashboard, no complex component tree needed. Smaller bundle.
- **Leaflet**: Mature, lightweight map library. CartoDB dark tiles match the dark theme perfectly.
- **Hand-rolled SVG charts**: No D3 needed for a simple area chart — saves ~500KB bundle.
- **Pipeline strategy**: BOM WISKI API is HTTP (not HTTPS), which would cause mixed-content errors in the browser. Pipeline runs server-side via GitHub Actions and writes JSON to the repo.
- **Seed data**: Embedded approximate values so the site shows something useful before the first pipeline run.
